# Discontinuous Galerkin Method - References

Repositório colaborativo para organização de artigos, livros, teses e materiais relacionados ao Método de Galerkin Descontínuo (DG).

O objetivo é manter uma base organizada de referências para estudo, desenvolvimento e aplicação do método.

---

## Organização
Os arquivos das referências estão separadas por tópicos:

| Pasta | Conteúdo |
|---|---|
| [Fundamentals](./Fundamentals) | Formulação básica, análise e teoria |
| Numerical Fluxes | Fluxos numéricos e estabilização |
| Applications | Aplicações diversas |
