# Galerkin Descontínuo (DG) 1D: Fundamentos e Implementação em Python

Bem-vindo ao repositório de estudos e implementação do **Método Galerkin Descontínuo (DG)** em uma dimensão.

Este projeto foi construído com uma abordagem Bottom-Up. O objetivo é desmistificar a matemática densa por trás dos métodos espectrais e traduzir conceitos teóricos complexos em código Python limpo e escalável. Aqui, você não encontrará "caixas pretas": cada integração, quadratura e matriz do método é deduzida, codificada e testada passo a passo.

### 🗂️ Estrutura do Repositório
Para facilitar o estudo e a navegação, o repositório está dividido em duas áreas principais:
* 📁 `Lessons/`: O coração do projeto. Contém os Jupyter Notebooks interativos com as aulas práticas, misturando deduções no texto e Live Debugging em Python. O sumário completo e a ordem recomendada de estudo encontram-se dentro desta pasta.
* 📁 `DG-references/`: A biblioteca de fundação. Contém anotações detalhadas, artigos de referência e os aprofundamentos teóricos que embasam as escolhas computacionais feitas nos notebooks.

### 🛠️ Tecnologias Utilizadas
Os scripts e notebooks foram desenvolvidos utilizando o ecossistema científico do Python para garantir performance e clareza matemática:

* Linguagem: Python 3.x
* Bibliotecas Principais:
  * `NumPy` (Cálculo matricial e manipulação de arrays)
  * `SymPy` (Matemática simbólica e dedução de sistemas)
  * `Matplotlib` (Visualização de dados e gráficos)
  * `Math` (Operações transcendentais nativas)
